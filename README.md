# ITMD567Project
A simple event management website with CRUD operations
### Tech

  - NodeJs
  - MongoDB
  - Pug
  - Mocha and Chai
  - Heroku - App live launch
### Installation

Install the dependencies, devDependencies. Change port (3000) and start the server.
Change port also @ bottom last line (index.js)

```sh
$ cd myapp
$ npm install 
$ node index.js
```
